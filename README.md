# excel-sql-projects

A portfolio of solo data analytics projects using Excel and SQL.

## What you'll find here
- Excel analyses (cleaning, PivotTables, summary dashboards)
- SQL analyses (aggregations, joins, business questions)
- Short write-ups explaining the business question, approach, and key insights

## Why this repository exists
This repo documents my progress and provides evidence of hands-on analytics skills through finished, reproducible projects.

## Project structure
Each project folder includes:
- `README.md` (business question, dataset, method, findings)
- Source file(s) or a link to the dataset
- Analysis files (Excel workbook, SQL scripts)
- Optional screenshots of final outputs
